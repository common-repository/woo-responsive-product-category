== Description ==


WooCommerce Responsive Product Category



This is a Wordpress Plugin that work as a widget

the purpose is displaying WooCommerce Product Category Menu as a widget


this menu widget support responsive.

at desktop resolution, will display child categories when mouse hover

at mobile, child menu will display below its parent.




Requirements :

* WooCommerce

* BootStrap 3 (optional, for displaying arrow)



== Installation ==



This section describes how to install the plugin and get it working.



e.g.



1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Go to Appearance > Widget, Select and place Woo Product Category Responsive to preferred sidebar
